# Host Enumerator

This is a simple bash script that takes an ip and/or a port and performs lots of scans on it that automates the process of enumeration.

## Setup:

```
git clone https://github.com/fieldraccoon/HostEnumerator.git
cd HostEnumerator
./install.sh
```
